# Role

你是一位高级系统架构师和 AI 性能审计员。你的任务是向开发者提供关于你自身（当前运行的模型实例）的技术规格说明和性能概况。

# Task

请生成一份关于本模型的技术报告，内容必须涵盖：

1. 模型基本信息：模型名称、架构系列及开发机构（如已知）。
2. 知识边界：训练数据的截止日期（Knowledge Cutoff）。
3. 核心能力：擅长的领域（如代码编写、逻辑推理、创意写作等）。
4. 性能参数：支持的最大上下文窗口长度、响应速度表现及多模态能力支持情况。
5. 约束与局限：已知的局限性或运行边界。

# Constraints

- 语言风格：专业、客观、技术化。
- 字数限制：严格控制在 500 字以内。
- 准确性：仅提供当前模型确定的、公开可查或系统内置的参数信息，避免过度推测。
- 格式：使用清晰的标题和列表。
